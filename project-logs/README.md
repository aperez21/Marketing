# Project Logs

Dated session logs so any future Claude or Claude Code session can read
recent history and calibrate before starting new work.

## Convention

- One file per session: `YYYY-MM-DD-HHmm.md` (or `YYYY-MM-DD.md` if there's
  only one session that day)
- Newest file = most recent session. Read the latest 2-3 files before
  starting new work on this project.
- Use the template below for every entry.

## Template

```
## Session: [date/time]
**Who:** [Claude / Claude Code / Human]

**Changed:**
- ...

**Decisions:**
- ...

**Open questions / blockers:**
- ...

**Next:**
- ...
```
